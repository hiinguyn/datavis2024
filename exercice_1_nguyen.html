Exercice no. 01

Ligne de temps faite avec Wikidata

Cette timeline montre des footballeurs français qui jouent en Premier League (Anglettere).

La requête est :

                #defaultView:Timeline
                SELECT ?UEFA_Champions_League ?UEFA_Champions_LeagueLabel ?date_of_birth ?place_of_birth ?place_of_birthLabel ?image WHERE {
                SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
                ?UEFA_Champions_League wdt:P118 wd:Q9448.
                ?UEFA_Champions_League wdt:P103 wd:Q150.
                OPTIONAL { ?UEFA_Champions_League wdt:P569 ?date_of_birth. }
                OPTIONAL { ?UEFA_Champions_League wdt:P19 ?place_of_birth. }
                ?UEFA_Champions_League wdt:P18 ?image. 
                }
                LIMIT 100
